# Extension Icons

Add the following PNG icons:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

For testing, you can use any placeholder images.

## Design Suggestions
- Simple gradient background (purple/blue like the popup)
- Letter "M" or chat bubble icon
- Keep it recognizable at small sizes

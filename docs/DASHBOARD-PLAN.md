### Phase 1: Core (2-3 days)
- [ ] Project setup (Vite + Tailwind)
- [ ] Auth pages (login, register)
- [ ] Main layout (sidebar, header)
- [ ] Models list page
- [ ] Model create/edit forms
- [ ] Persona dropdown

### Phase 2: Fanvue (1-2 days)
- [ ] Connect Fanvue button + modal
- [ ] OAuth flow integration
- [ ] Platform status display
- [ ] Basic chats list (read-only)

### Phase 3: Chats (2-3 days)
- [ ] Chat inbox UI
- [ ] Message history display
- [ ] AI suggestion component
- [ ] Send message (Fanvue)

### Phase 4: Polish (1-2 days)
- [ ] Team management page
- [ ] Basic analytics cards
- [ ] Error handling
- [ ] Loading states

**Total:** ~8-10 days
